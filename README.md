# Setting Up (npm)
1. Have Node 12 LTS or greater installed
2. `npm install -g expo-cli`
3. `npm i -g create-react-native-app`
4. `git clone https://github.com/limsiying/ace` and cd into the project directory
5. `npm start`

# Run App
1. `cd ace`
2. `expo start`
